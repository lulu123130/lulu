# 基于 Vue 3 + Vite + Element-plus 的框架

| 依赖         | 版本    | 功能     |
| ------------ | ------- | -------- |
| nodjes       | >22.12     |          |
| vue          | 3.5.14  | 框架     |
| vite         | 7.1.17  | 构建工具 |
| vue-router   | 4.5.1   | 路由     |
| pinia        | 3.0.2   | 状态管理 |
| element-plus | 2.9.10  | UI组件   |
| axios        | 1.9.0   | http请求 |
| echarts      | 5.5.0   | 图表     |
| animate.css  | 4.1.1   | 动画样式 |
| dayjs        | 1.11.11 | 时间处理 |
| sass | 1.93.2 | css预处理 |

##### 1. 建议使用vscode编辑器 + Vue.volar + esbenp.prettier-vscode

``````json
{
  "recommendations": [
    // Vue 3 的语言支持
    "Vue.volar",
    // 使用 Prettier 的代码格式化程序
    "esbenp.prettier-vscode"
  ]
}
``````

##### 2. 命令
``` bash
# 安装依赖
npm install

# 启动
npm run serve

# 打包
npm run build
```
